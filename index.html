The game is afoot

Let's use Watson's Dialog API to gather the facts of the case. We'll apply these facts to Watson's Concept Insights API to find connections. All of the objects and clues can be stored together in Hackpad.

// An Example

Usage of the Concept Insights API has already been proven with some manual queries. Objects such as a coin, key, letter, ring, pocket watch, and towel can be related to single concepts in Wikipedia. Actually, many of these objects can have multiple meanings, but we'll stay with the physical objects in this example.

What concept connects all of these objects? A Chain

  • Coins are often an ornament on a necklace (chain).
  • Keys are carried on a keychain.
  • Letters can be chain letters.
  • Rings are links of a chain.
  • Pocket watches are attached to a chain.
  • Towels are chains of thread.
I'll admit that isn't very useful as a clue, but imagine if some of these objects were people and places. Luckily, there is tons of Sherlock Homes data in Wikipedia!
